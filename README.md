# 📊 Startup Funding Analysis Dashboard

## 🚀 Project Overview
The **Startup Funding Analysis Dashboard** is a data-driven project designed to analyze startup investments and uncover meaningful insights into funding trends, investor behavior, and market growth.

This project transforms raw, unstructured data into actionable insights through **data cleaning, analysis, and visualization**.

---

## 🎯 Project Objectives
- Analyze startup funding trends over time  
- Identify top-funded sectors and cities  
- Understand investor patterns  
- Build a clean and reliable dataset for analysis  
- Present insights using clear visualizations  

---

## 👨‍💻 My Contribution (Data Cleaning & Preparation)
I was primarily responsible for the **Data Cleaning & Preprocessing phase**, which is the most critical part of any data analysis project.

### 🔥 Why This Part is Important
Real-world datasets are messy and unreliable. Without proper cleaning:
- Analysis becomes inaccurate  
- Visualizations become misleading  
- Decision-making becomes flawed  

My work ensured the dataset became **accurate, consistent, and analysis-ready**.

---

## 🧹 Data Cleaning (Core & Most Important Part)

### 1️⃣ Handling Missing Values
- Identified missing/null values across all columns  
- Applied appropriate strategies:
  - **Numerical columns** → filled using median/mean  
  - **Categorical columns** → filled using mode  
  - Dropped rows where data was insufficient  

👉 Result: Eliminated incomplete data and improved dataset reliability  

---

### 2️⃣ Removing Duplicate Records
- Detected duplicate entries using Pandas  
- Removed redundant rows  

👉 Result: Prevented data bias and duplication errors  

---

### 3️⃣ Data Standardization
- Cleaned column names (lowercase, no spaces)  
- Standardized text fields (city, startup names, investors)  
- Removed extra spaces and inconsistencies  

👉 Result: Uniform dataset for easy processing  

---

### 4️⃣ Data Type Conversion
- Converted date columns into proper datetime format  
- Converted funding amounts into numeric format  

👉 Result: Enabled accurate time-based and numerical analysis  

---

### 5️⃣ Handling Inconsistent Data
- Fixed inconsistent naming (e.g., city variations)  
- Removed unwanted characters and symbols  
- Unified categorical values  

👉 Result: Improved data integrity and consistency  

---

### 6️⃣ Outlier Detection & Handling
- Used **IQR (Interquartile Range)** method  
- Removed extreme values that could distort analysis  

👉 Result: More realistic and meaningful insights  

---
